# Story-013-error-book: Error Book System

**Phase**: Phase 3: Question Bank
**Goal**: 错题自动收集与复习
**预估时间**: 4-6 Hours
**Story Points**: 5
**前置依赖**: Story-012
**负责人**: _待分配_

---

## 1. Objectives (实现目标)

- [ ] 在判卷逻辑中，自动将错题写入 `ErrorBook` 表。
- [ ] 实现错题本列表页，支持按学科筛选。
- [ ] 实现 '消灭错题' 功能 (重做正确后移除)。

---

## 2. Tech Plan (技术方案)

- DB: `upsert` 逻辑 (如果已存在错题则更新次数)。
- API: 获取错题列表。
- UI: 错题卡片，带有 '移除' 按钮。
