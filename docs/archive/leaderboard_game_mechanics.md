# 排位赛核心游戏机制 (Leaderboard Game Mechanics) v1.2

为了打造“飞轮效应”，激励学生持续学习、互动并贡献内容，LearnMore 排位赛遵循 "S.P.A.K." 系统设计。v1.2 版本在 UGC 生态中强化了轻量级交互激励。

## 1. 赛季周期 (Cycle) - 双周赛制

*   **设定：** **“双周赛季” (Bi-Weekly League)**
*   **时间：** 每两周为一个完整赛季。Day 1 (周一) 00:00 开启，Day 14 (次周周日) 23:59 结算。
*   **季中活动 (Mid-Season Mania):** 第一周周末 (Sat-Sun) 特定任务开启 1.5倍 XP，提供追分机会。
*   **结算逻辑：** Seasonal XP 清零，Lifetime XP 累积。

## 2. 段位体系 (Tier Progression)

*   **段位 (6阶):** Bronze, Silver, Gold, Platinum, Diamond, Challenger (Top 100).
*   **升降级:** 晋级区 (Top 15%), 保级区 (Middle 65%), 降级区 (Bottom 20%)。

## 3. XP 获取途径 (The Action) - 引入全方位生态激励

### A. 个人成长 (Personal Growth) - 基础分
| 行为类型 | 行为描述 | XP 奖励 | 目的 |
| :--- | :--- | :--- | :--- |
| **学习新知** | 完成 Video Lesson | +50 XP | 基础进度 |
| **巩固练习** | Chapter Quiz 满分 | +30 XP | 追求质量 |
| **消灭弱点** | **Error Book (消灭错题)** | **+40 XP** | **核心驱动：补短板** |
| **每日坚持** | 每日打卡 Streak | +20 XP | 活跃度 |

### B. UGC 深度贡献 (Deep Contribution) - 高价值
*鼓励优质内容输出，实行“重赏”。*

| 行为类型 | 行为描述 | XP 奖励 | 核心目的 |
| :--- | :--- | :--- | :--- |
| **神回复** | **回答问题被采纳 (Accepted)** | **+150 XP** | **鼓励学霸扶贫** |
| **优质提问** | 提问获 3+ 点赞 | +30 XP | 鼓励不懂就问 |
| **知识分享** | 分享笔记/Cheat Sheet | +50 XP | 扩充内容库 |
| **被动收入** | 内容每获 10 赞/收藏 | +20 XP | 鼓励长尾优质内容 |

### C. 社区交互参与 (Community Engagement) - 活跃分
*鼓励普通用户参与社区筛选和讨论，增加社区热度。*

| 行为类型 | 行为描述 | XP 奖励 | 限制与目的 |
| :--- | :--- | :--- | :--- |
| **慧眼识珠** | **点赞/评价 (Upvote/Rate)** | **+2 XP/次** | 上限 10 XP/天。鼓励筛选好内容。 |
| **参与讨论** | **发表评论 (Comment)** | **+5 XP/次** | 上限 20 XP/天。鼓励发表观点和讨论，不仅是给答案。 |
| **投票参与** | 参与社区 Poll 投票 | +5 XP | 增加参与感。 |

*注：设置每日上限 (Daily Cap) 是为了防止用户恶意刷屏刷分。*

### D. 赛季特殊事件 (Season Events)
每个赛季随机激活一种“词缀”：
1.  **Sniper Season:** 错题 XP x2.
2.  **Helper Season:** 社区回答 XP x2.
3.  **Debate Season:** 社区评论 XP x2.

## 4. 飞轮效应总结
1.  **Input:** 学习、刷题。
2.  **Interaction:** 点赞、评论、投票（获取活跃分）。
3.  **Contribution:** 提问、回答、写笔记（获取高额分）。
4.  **Reward:** 排名上升，段位晋升。
5.  **Retention:** 社交互动增加用户粘性。
